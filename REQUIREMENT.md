# MONITA SISTEM REKOMENDASI

## SERVER
- ubuntu server > 20.04 LTS
- koneksi ssh
- disk 100GB
- RAM 8GB
- cpu 4 core

## SOFTWARE PENDUKUNG
- nginx/haproxy
- nvm versi terakhir
- nodejs versi > 20 (install melalui nvm)
- mysql versi v5.7 / v8 (gunakan mysql_native_password)
- pm2 (install melalui npm)
- redis versi > 6.0
